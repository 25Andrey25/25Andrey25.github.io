MyDevPortfol.io

Веб-приложение, помогающее разработчикам создавать красивое оформленные портфолио за считанные минуты

ТЕХНОЛОГИЯ

Mydevportfol.io построен с использованием Angular2 +, с серверной частью узла для обработки запросов API и работает на Heroku. Платежи обрабатываются через Stripe, Firebase предоставляет решение для работы с базами данных, а Bootstrap3 является предпочтительным CSS-фреймворком для этого проекта

Jobb.ie

Простое веб-приложение, созданное для быстрого поиска работы в Ирландии. Пользователи вводят поисковый запрос и желаемый округ, и им предлагаются списки вакансий с 3 самых популярных сайтов по трудоустройству в Ирландии

Jobbie использует Bootstrap3 для макета, ориентированного на мобильные устройства, с jQuery для удобства манипулирования страницами. На серверной части Express/Node.js приложение работает на Heroku dyno, который действует как промежуточное программное обеспечение для очистки сайтов вакансий и возврата информации пользователю.

My Delivery Pal

Android-приложение для водителей доставки и курьеров, позволяющее им выполнять свою работу быстрее и эффективнее

Технология

Delivery Pal построен на базе Ionic, Angular и Firebase для отслеживания местоположения в режиме реального времени. Интеграция с Google Maps API для поиска адресов и навигации. А Node.js сервер, размещенный на Heroku, действует как посредник, обрабатывая внутреннюю логику и обновления базы данных в Firebase

Vistory

Мобильное автомобильное приложение

Android-приложение для владельцев транспортных средств, позволяющее быстро и легко регистрировать расходы на техническое обслуживание и запчасти для своих автомобилей.

Технология

Vistory построена на платформе Ionic, которая использует Angular.js фреймворк для создания кроссплатформенных мобильных приложений. На серверной части Vistory использует Express/Node.js приложение на Heroku для обработки учетных записей пользователей и аутентификации, в то время как удаленная база данных NoSQL находится на Cloudant для резервного копирования пользовательских данных

Slack Chat Bot

Чат-бот Slack, созданный для быстрой проверки доступности доменных имен в самых популярных TLD.

Технология

Бот Slack взаимодействует с Node.js приложение, работающее на Heroku, которое запрашивает информацию из бесплатного api для проверки домена, анализирует ответ и отправляет клиенту ответ с цветовой кодировкой.
